# Data Structures and Algorithms in C ⚙️

This repository contains my handwritten C code and implementations of core Data Structures and Algorithms, following the renowned **[DSA in Hindi by Code With Harry](https://www.youtube.com/playlist?list=PLu0W_9lII9ahIappRPN0MCAgtOu3lQjQi)** playlist.

## 📌 About The Repository
This repository serves as a practical log of my journey mastering memory management, pointers, and foundational algorithms in C. It contains "from scratch" implementations of data structures without the use of standard template libraries, ensuring a deep, under-the-hood understanding of how computer memory works.

## 💻 Tech Stack
* **Language:** C
* **Compiler:** GCC
* **Topics:** Core DSA, Pointers, Dynamic Memory Allocation (`malloc`, `calloc`, `free`)

## 📚 Topics Mastered (Current Progress)
This repository covers implementations up to **Binary Search Tree Insertion**. 

**1. Arrays & Abstract Data Types (ADT)**
* Creation and Operations (Traversal, Insertion, Deletion)
* Linear Search vs. Binary Search

**2. Linked Lists**
* Singly Linked Lists (Insertion, Deletion at various positions)
* Circular Linked Lists
* Doubly Linked Lists

**3. Stacks**
* Implementation using Arrays and Linked Lists
* Applications: Parenthesis Matching, Multiple Parenthesis Matching
* Infix to Postfix Conversion using Stack

**4. Queues**
* Implementation using Arrays and Linked Lists
* Circular Queues
* Double Ended Queues (DEQueue)

**5. Trees (Current Focus)**
* Binary Tree Representation (Linked Representation)
* Tree Traversals: Preorder, Postorder, and Inorder
* Binary Search Trees (BST): Checking if a tree is a BST
* BST Operations: Searching (Iterative & Recursive) and **Insertion**
